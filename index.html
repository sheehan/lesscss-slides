Some features of LESS

1. Nested Rules

#header {
  color: black;
  .navigation {
    font-size: 12px;
  }
  .logo {
    width: 300px;
    &:hover { text-decoration: none }
  }
}
  • Provide organization
  • Mirrors HTML structure

2. Variables

@color: #4D926F;

#header {
  color: @color;
}
h2 {
  color: @color;
}
  • Variables are constant
  • Variables have block-level scope

3. Mixins

.rounded-corners (@radius: 5px) {
  border-radius: @radius;
  -webkit-border-radius: @radius;
  -moz-border-radius: @radius;
}

#header {
  .rounded-corners;
}
#footer {
  .rounded-corners(10px);
}

4. Operations

@the-border: 1px;
@base-color: #111;
@red:        #842210;

#header {
  color: @base-color * 3;
  border-left: @the-border;
  border-right: @the-border * 2;
}
#footer {
  color: @base-color + #003300;
  border-color: desaturate(@red, 10%);
}

5. Functions

@the-border: 1px;
@base-color: #111;
@red:        #842210;

#header {
  color: @base-color * 3;
  border-left: @the-border;
  border-right: @the-border * 2;
}
#footer {
  color: @base-color + #003300;
  border-color: desaturate(@red, 10%);
}

Using LESS in Grails

LESS in Grails

grails install-plugin lesscss
            
<html>
<head>
    ...
    <less:stylesheet name="your_stylesheet" />
    <less:scripts />
</head>

LESS in Grails using Resources Plugin

grails install-plugin lesscss-resources
            
'style' {
    resource url:'less/test.less', attrs:[rel: 'stylesheet/less', type:'css']
    resource url:'css/normal.css'
    resource url:'css/normal2.css'
}
