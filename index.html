What is LESS?

  • CSS Preprocessor
  • Extenstion of CSS
  • CSS is valid LESS
  • Compiles to CSS
  • TODO diagram

Why should I care?

  • CSS is repetitive
  • CSS is flat
  • Like writing programs in byte code

LESS addresses these issues by adding dynamic functionality and structure.

Some features of LESS

Nested Rules

#header {
  color: black;
  .navigation {
    font-size: 12px;
  }
  .logo {
    width: 300px;
    &:hover { text-decoration: none }
  }
}
  • Provide organization
  • Mirrors HTML structure

Variables

@color: #4D926F;

#header {
  color: @color;
}
h2 {
  color: @color;
}
  • Variables are constants
  • Variables have block-level scope

Mixins

  • Mix-in properties from another ruleset 
    .bordered {
  border-top: dotted 1px black;
  border-bottom: solid 2px black;
}

#menu a {
  color: #111;
  .bordered;
}

.post a {
  color: red;
  .bordered;
}
  • Parametric 
    .rounded-corners (@radius) {
  border-radius: @radius;
  -webkit-border-radius: @radius;
  -moz-border-radius: @radius;
}

#footer {
  .rounded-corners(10px);
}
  • Default values 
    .rounded-corners (@radius: 5px) {
  border-radius: @radius;
  -webkit-border-radius: @radius;
  -moz-border-radius: @radius;
}

#header {
  .rounded-corners;
}

#footer {
  .rounded-corners(10px);
}
  • @arguments 
    .box-shadow (@x: 0, @y: 0, @blur: 1px, @color: #000) {
  box-shadow: @arguments;
  -moz-box-shadow: @arguments;
  -webkit-box-shadow: @arguments;
}

.box-shadow(2px, 5px);
  • Guards 
    .mixin (@a) when (lightness(@a) >= 50%) {
  background-color: black;
}

.mixin (@a) when (lightness(@a) < 50%) {
  background-color: white;
}

.mixin (@a) {
  color: @a;
}

.class1 { .mixin(#ddd) }

.class2 { .mixin(#555) }

Operations

  • Operations on units 
    @border: 1px;
@thicker-border: @border + 3px;

#header {
  border-left: @border;
  border-right: @border * 2;
  border-top: @thicker-border;
}

#footer {
  border: @thicker-border solid black;
}
  • Operations on colors 
    @base-color: #111;

#header {
  color: @base-color * 3;
}
#footer {
  color: @base-color + #003300;
}

Functions

  • Math functions 
    .example {
  width:   round(50px / 3);
  margin:  ceil(5em / 3);
  padding: floor(10in / 4);
  height:  percentage(1 / 5);
}
  • Color functions 
    @base-color: #111;
.example {
  color:               @base-color;
  border-top-color:    lighten(@base-color, 20%);
  border-right-color:  darken(@base-color, 20%);
  border-bottom-color: saturate(@base-color, 20%);
  border-left-color:   desaturate(@base-color, 20%);
  background-color:    fade(@base-color, 20%);
}

Using LESS in Grails

LESS in Grails

grails install-plugin lesscss
            
<html>
<head>
    ...
    <less:stylesheet name="your_stylesheet" />
    <less:scripts />
</head>

LESS in Grails using Resources Plugin

grails install-plugin lesscss-resources
            
'style' {
    resource url:'less/test.less', attrs:[rel: 'stylesheet/less', type:'css']
    resource url:'css/normal.css'
    resource url:'css/normal2.css'
}
